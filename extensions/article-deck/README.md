# Article Deck

A Chrome extension for generating slides from web articles.

## Features

- Extract article content from web pages
- Generate Marp-format slides using Gemini API
- Preview and download slides

## Development

```bash
bun install
bun run build
```

## Getting Started

1. Download the release zip file or build locally
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `dist/` folder
5. Set your Gemini API key in the extension options
